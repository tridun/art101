Lab 4: Pseudocoding and Problem-solving

Challenge

The overall challenge of this assignment was to get used to the idea of writing pseudocode by breaking down problems into high-level instructions.

Problems

For Task 1 of this lab we created pseudocode for an everyday task. We chose to consult online recipes own personal experience to create instructions for making lemonade.

For Task 2 we created pseudocode for a computer task. We chose to write pseudocode for a rock-paper-scissors game.

For Task 3 we turned the pseudocode for the computer task into comments.

We encountered few issues while completing these tasks but had a little difficulty stream-lining our lists to be simple yet descriptive.

Results

Task 1: Making Lemonade

  1. Place pitcher on counter
  2. Cut 6 seedless lemons in half
  3. Squeeze all lemon halves into pitcher
  4. Pour 3 cups of cold water into the pitcher
  5. Add one cup of granulated white sugar into the pitcher
  6. Mix all ingredients thoroughly with spoon
  7. Add ice cubes to glass
  8. Pour serving of lemonade into glass

Task 2: Creating a Rock Paper Scissors Game

  1. Create a list of play options
    1. Rock, Paper, Scissors
  2. Have the computer randomly select one of the list options
    1. Use randomizer function for play options 0-2
  3. Ask for user input
  4. Test for win condition/tie
    1. Use if/else if/else statements to compare player and computer inputs
